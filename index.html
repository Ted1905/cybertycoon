Temel Ağ Komutları

  • nmap [IP]: Hedef IP'deki açık portları hızlıca tarar.
  • nmap -sV [IP]: Açık portlardaki servislerin versiyonunu tespit eder.
  • netstat -antp: Aktif bağlantıları ve açık portları listeler (Backdoor avı için).
  • ifconfig eth0 down: İnternet bağlantısını anında keser (Fidye yazılımı için).

Sızma (Red Team) Araçları

  • sqlmap -u "URL" --dbs: SQL Injection açıklarını sömürür.
  • dirb [URL]: Gizli admin veya yedek klasörlerini zorla (brute-force) bulur.
  • hydra -L [user] -P [pass] ftp://[IP]: Kaba kuvvet saldırısı yapar.
  • john --wordlist=[dosya] hash.txt: Şifrelenmiş (hash) metinleri kırmaya çalışır.
  • nc -lvnp 4444: Reverse shell için port dinler.

Savunma (Blue Team) Komutları

  • iptables -A INPUT -s [IP] -j DROP: Belirli bir IP'den gelen trafiği engeller.
  • find / -perm -4000: SUID yetkili dosyaları arar.
  • cat /var/log/auth.log: Linux giriş (SSH) log dosyasıdır.

İpucu: Terminalde TAB tuşuna basarak komutu otomatik tamamlayabilirsiniz.
