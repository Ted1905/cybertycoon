### 🇹🇷 Türkçe 
**CyberTycoon 2026**, siber güvenlik dünyasını hem savunma (Blue Team) hem de saldırı (Red Team) perspektifinden deneyimlemenizi sağlayan, tarayıcı tabanlı bir simülasyon ve eğitim oyunudur. 

[cyberakademi.org](https://cyberakademi.org) vizyonuyla, sektöre yeni adım atacaklara siber güvenlik jargonu ve temel komutları öğretmek amacıyla geliştirilmiştir.

#### 🚀 Oyunu Oyna
👉 **[OYNAMAK İÇİN BURAYA TIKLAYIN](https://ted1905.github.io/cybertycoon)**

#### 🎮 Özellikler
* **Mavi Takım (Savunma):** Oltalama (Phishing) e-postalarını analiz edin, güvenlik duvarı kurun ve Ransomware saldırılarını durdurun.
* **Kırmızı Takım (Saldırı):** Dark Web araçları edinin, Nmap ile zafiyet tarayın, SQL Injection ve Brute-Force simülasyonları yapın.
* **Terminal Simülasyonu:** Gerçekçi Linux komutlarıyla (iptables, netstat vb.) olaylara müdahale edin. (`TAB` tuşu ile otomatik tamamlama mevcuttur!)
* **OPSEC ve İtibar:** Yakalanma riskinizi yönetin. FBI operasyonundan kaçının!

### 🇬🇧 English
**CyberTycoon 2026** is a browser-based simulation and educational game that allows you to experience the cybersecurity world from both defensive (Blue Team) and offensive (Red Team) perspectives.

Developed with the vision of [cyberakademi.org](https://cyberakademi.org), it aims to teach cybersecurity jargon and basic commands to newcomers in the industry.

#### 🚀 Play the Game
👉 **[CLICK HERE TO PLAY](https://ted1905.github.io/cybertycoon)**

#### 🎮 Features
* **Blue Team (Defense):** Analyze phishing emails, configure firewalls, and stop Ransomware attacks immediately.
* **Red Team (Offense):** Acquire Dark Web tools, scan for vulnerabilities with Nmap, and simulate attacks like SQL Injection and Brute-Force.
* **Terminal Simulation:** Respond to incidents using realistic Linux commands (iptables, netstat, etc.). (Use the `TAB` key for auto-complete!)
* **OPSEC & Reputation:** Manage your risk of getting caught. Avoid an FBI raid!
